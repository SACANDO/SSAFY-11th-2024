# git lab

```java
**-- 마스터 브랜치에서 작업하는 경우 --**
git clone 본인깃주소
cd 폴더
폴더를 eclipse에서 import한 후, 실습 문제 풀기
**git add .
git commit -m "커밋메시지"
git push**
실습 제출하기

**-- 새로 브랜치를 파서 작업하는 경우 --**
git clone 본인깃주소
cd 폴더
git branch sj => 브랜치를 새로 만듦
git checkout sj => 브랜치를 옮김
git add . => 변경사항을 커밋 리스트에 올리고
git commit -m "커밋메시지"
git push -u origin sj => 서버에 sj라는 브랜치를 새로 만들고 
그 브랜치에다가 로컬 sj 브랜치의 작업 내용을 복사

=> 서버에 sj 브랜치가 올라간 상황

merge request 만들기(팀원)
Merge requests 탭에 가서 new merge request
Source : sj  -> target : master

merge하기(팀장)
sj 브랜치의 작업내용이 master에 반영되고
sj 브랜치는 사라짐!
```
