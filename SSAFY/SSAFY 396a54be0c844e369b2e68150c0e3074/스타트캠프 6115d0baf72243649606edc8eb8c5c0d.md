# 운영체제와 프로그램

- 프로그램 :
- 운영체제(OS) : 시스템 하드웨어를 관리할 뿐 아니라 응용 소프트 웨어를 실행하기 위하여 **하드웨어 추상화 플랫폼(키보드, 마우스)**과 **공통 시스템 서비스(메모리, 파일시스템 관리)**를 제공하는 시스템 소프트웨어
- 1byte : 컴퓨터 정보 처리의 최소단위 2^8=256가지
- 2의 보수
    - 맨앞에 1 : 음수, 0: 양수
        1. 비트를 뒤집는다
        2. 1을 더한다
    - 256에서 빼고 - 붙여도 됨

# 프로그램과 논리/증명

- 가정이 거짓이면 명제는 참이다.
    - 19세 미만은 맥주를 마실 수 없다. 기억!!
    
    → **19세 이상이면 가정이 거짓** →마셔도 참 안 마셔도 참 → **즉 무조건 참**
    
    → 19 미만이면 가정이 참 → 안 마셔야 참! → 즉 안마셨는지 확인해야 함
    
    → **결과가 참이면 가정을 확인하지 않아도 된다.** 가정이 참이어도 참 거짓이어도 참 → **즉 무조건 참**
    
- 수학적 귀납법 : P(1)이 참이고 P(n)→P(n+1)이 참이면 참
    
     여기서는 P(n)이 참인것을 증명할 필요가 없고 P(n)→P(n+1)이 참임을 증명해야 한다.
    
    → P(n)이 참임을 가정해놓고 P(n+1)이 참이면 참이다 : P(n)을 블랙박스로 볼 때 편함

    public static void captureAndCombineScreenshots(WebDriver driver1, WebDriver driver2, WebDriver driver3, String savePath) throws IOException {
        // 각 브라우저 창에서 스크린샷 캡처
        BufferedImage img1 = ImageIO.read(((TakesScreenshot)driver1).getScreenshotAs(OutputType.FILE));
        BufferedImage img2 = ImageIO.read(((TakesScreenshot)driver2).getScreenshotAs(OutputType.FILE));
        BufferedImage img3 = ImageIO.read(((TakesScreenshot)driver3).getScreenshotAs(OutputType.FILE));

     // 결합할 이미지의 총 크기 계산
        int width = img1.getWidth() + img2.getWidth() + img3.getWidth();
        int height = Math.max(Math.max(img1.getHeight(), img2.getHeight()), img3.getHeight());

        // 새 이미지 생성
        BufferedImage combined = new BufferedImage(width, height, BufferedImage.TYPE_INT_ARGB);
        Graphics g = combined.getGraphics();

     // 이미지 결합
        g.drawImage(img1, 0, 0, null);
        g.drawImage(img2, img1.getWidth(), 0, null);
        g.drawImage(img3, img1.getWidth() + img2.getWidth(), 0, null);

        // 파일로 저장
        ImageIO.write(combined, "PNG", new File(savePath));
    }
